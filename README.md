# Crypto Custody Landscape

A comprehensive guide to digital asset custody for institutional decision-makers, published by the Enterprise Ethereum Alliance.

## Overview

This static website provides an educational resource covering:

- **What is Crypto Custody** - Fundamentals of private key management and digital asset control
- **Why Custody Matters** - Fiduciary obligations, regulatory requirements, and operational risks for institutions
- **Custody Models** - Self-custody, hybrid/shared, and third-party custody approaches with trade-offs
- **Decision Framework** - Structured evaluation process for selecting custody solutions
- **Provider Landscape** - Categorization of custody providers by operating model (not marketing labels)
- **Control Stack** - Technical architecture layers from physical security to governance policies
- **Institutional Use Cases** - Profiles for asset managers, corporate treasury, exchanges, banks, DAOs, and HNWIs
- **Methodology & Neutrality** - Transparent approach and limitations of this resource

## Deployment

This site is designed for GitHub Pages. To deploy:

1. Push to the `main` branch
2. Enable GitHub Pages in repository settings (Settings > Pages)
3. Select `main` branch as the source
4. The site will be available at `https://<username>.github.io/<repo-name>/`

No build process is required - the site uses static HTML, CSS, and JavaScript.
